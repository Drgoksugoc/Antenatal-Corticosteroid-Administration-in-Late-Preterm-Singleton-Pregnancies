# Manuscript–Repository Alignment Checklist

This repository version corresponds to the revised manuscript after peer review.

## Required manuscript alignment

Please confirm that the revised manuscript includes the following items before resubmission:

1. ACS dose/timing data are described in the Results or Supplementary Materials:
   - Single documented dose: 108/126 (85.7%)
   - Double documented dose/course: 18/126 (14.3%)
   - Delivery within 24 hours: 70/126 (55.6%)
   - Delivery between 24 hours and 7 days: 48/126 (38.1%)
   - Delivery after >7 days: 8/126 (6.3%)

2. The limitation statement should NOT say that dose completeness or ACS-to-delivery timing were unavailable. Instead, state that labor induction, antibiotic administration, active vs expectant PROM management, and total hospital length of stay were not reliably available.

3. The primary outcome footnote should state:
   "Primary endpoint: documented initial respiratory support at birth, derived from the neonatal resuscitation and oxygen-support category fields. It included hood oxygen, CPAP, PBV/positive-pressure ventilation, or intubation."

4. The Data Availability statement should match the public content of this repository. If the de-identified dataset is uploaded here, use:
   "De-identified data, analysis code, and derived outputs are publicly available at: [GitHub URL]."

5. If the de-identified dataset is not uploaded until acceptance, remove the data CSV before public upload and use:
   "Analysis code and derived outputs are publicly available at: [GitHub URL]. De-identified data will be deposited in a public repository upon acceptance."
