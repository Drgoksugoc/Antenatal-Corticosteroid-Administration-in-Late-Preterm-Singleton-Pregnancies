# Data use statement

The dataset in this repository is de-identified and is provided to support transparency and reproducibility of the associated manuscript.

Permitted use:
- Reproducing the analyses in the manuscript.
- Scholarly review, teaching, and non-commercial methodological evaluation with citation.

Restrictions and expectations:
- Do not attempt to re-identify any individual.
- Do not link the dataset to external records for re-identification.
- Do not redistribute modified datasets in a way that implies endorsement by the authors.
- Cite the associated article and this repository when using the data or code.

The code is released under the MIT License. The clinical dataset is shared for scholarly reproducibility; contact the corresponding author for questions about reuse beyond reproducing the manuscript analyses.
