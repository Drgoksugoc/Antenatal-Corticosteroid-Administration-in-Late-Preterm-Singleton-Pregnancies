# Data

`ACS_Late_Preterm_deidentified.csv` is the de-identified source dataset used by the R analysis pipeline.

Rows: 1091
Columns: 36

The main analytic cohort is derived by restricting to 34+0 to 36+6 weeks and excluding pregnancies with ACS before 34 weeks.

The dataset contains a pseudonymized `ID_deidentified` field for reproducibility. It does not contain names, exact dates, addresses, telephone numbers, or medical record numbers.

See `../docs/data_dictionary.csv` and `../docs/CODEBOOK.md` for variable definitions.
